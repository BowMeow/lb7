# Тема лабораторной работы: Реализация метода рекурсивного спуска для синтаксического анализа.

## Цель работы: Разработать для грамматики алгоритм синтаксического анализа на основе метода рекурсивного спуска.

### Задание: В соответствии с вариантом для заданной грамматики необходимо разработать и реализовать алгоритм анализа на основе метода рекурсивного спуска.
Вариант 2
Для грамматики G[<условный оператор>] разработать и реализовать
алгоритм анализа на основе метода рекурсивного спуска.
G[<условный оператор>]:
1. <условный оператор> ::= IF <условие> THEN <оператор>
2. <условие> ::= <выражение> <операция отношения>
<выражение>
3. <выражение> ::= <терм> {+ <терм>}
4. <терм> ::= <множитель> {* <множитель>}
5. <множитель> ::= < идентификатор > | (<выражение>)
6. <идентификатор> ::= <буква> {<буква> | <число>}
7. <оператор> ::= <выражение> | <условный оператор>
<операция отношения> ::= ”==” | ”<” | ”<=” | ”>” | ”>=” | ”!=”
<буква> ::= a | b | c | … | z | A | B | … | Z
<число> ::= <цифра> {<цифра>}
<цифра> 0 | 1 |...| 8 | 9
